1. Qual o retorno para AvaliarDesempenho(7.5)?

string AvaliarDesempenho(double nota) { if (nota >= 9) return "Excelente"; else if (nota >= 7) return "Bom"; else if (nota >= 5) return "Regular"; else return "Insuficiente"; }
  • Excelente
  • Bom
  • Regular
  • Insuficiente

2. O que as "Cláusulas de Guarda" (Early Return) evitam?

  • O uso de variáveis locais.
  • O "Código Espaguete" (excesso de ifs aninhados).
  • A criação de métodos na classe.

3. Por que usamos métodos em C#?

  • Para reaproveitar a lógica e evitar repetições desnecessárias.
  • Para tornar a execução do programa mais lenta.
  • Porque estruturas de condição não funcionam sozinhas.
